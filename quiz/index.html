Quiz yourself.

Three category quizzes — Rust basics, this site's Rust, and quant finance basics. Multiple choice, one question at a time, explanations after each answer with links back to the curriculum. Progress and scores persist locally; you can retake any quiz from scratch.

Categories

  1. 01 Rust basics Ownership, borrowing, modules, enums, the type system. Every question maps onto a pattern this site's own crate uses.
  2. 02 This site's Rust Specific to the code in crates/backtest-core/. If you've read the source, you'll recognize every answer.
  3. 03 Quant finance basics Sharpe, drawdown, hit rate, regimes, the no-lookahead constraint. Things you'd want to know before reading any backtest result.

How the quiz works

One question at a time. Click an option to lock in your answer; an explanation appears with the correct choice highlighted. Move to the next question with the button at the bottom. After the last question you'll see a score and a per-question recap. Your progress is stored in your browser; clearing site data resets it.
