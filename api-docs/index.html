Reference

HTTP API reference.

The selectsectors.com backend is nine Netlify Functions rooted at /api/. Most are reads (GET) for status and roster data; three are writes (POST) that consume rate- limit slots and dispatch a background WASM run. This page is the single source of truth for what each endpoint accepts and returns. The canonical implementation lives at netlify/functions/.

Conventions

  • All endpoints accept and return application/json unless noted.
  • Origin is https://selectsectors.com. The /api/* prefix is rewritten to /.netlify/functions/* by netlify.toml.
  • Rate limit is per-IP, two rolling windows: backtest at 2/hour and 5/day; chat at 30/hour and 100/day.
  • Cache hits on /api/backtest, /api/compare, and /api/scan bypass rate limit (deterministic re-runs of an already-cached result cost nothing on the backend).
  • Long backtests dispatch to a Netlify Background Function (15-min wall clock) and return 202 Accepted with a content hash; poll /api/result?hash=<hash> for completion.

Read endpoints (no rate limit)

GET /api/health

Operational health check. Reports the deployed version, commit, deploy ID, the catalog of WASM strategies, and the configured rate-limit caps for both backtest and chat.

{
  "status": "ok",
  "site": "selectsectors.com",
  "version": "0.1.2",
  "commit": "<git sha>",
  "deployId": "<netlify deploy id>",
  "timestamp": "2026-05-18T05:30:00.000Z",
  "strategies": [
    "buy_and_hold",
    "sma_crossover",
    "momentum",
    "rsi_mean_reversion",
    "breakout",
    "bollinger_bands"
  ],
  "rateLimits": {
    "backtest": { "hour": 2, "day": 5 },
    "chat":     { "hour": 30, "day": 100 }
  }
}

GET /api/universe

The 23-symbol roster. SPX index, SPY broad-market ETF, eleven SPDR sector ETFs, eleven anchor single names.

{
  "categories": {
    "index":   ["SPX"],
    "broad":   ["SPY"],
    "sectors": ["XLB", "XLC", "XLE", "XLF", "XLI", "XLK",
                "XLP", "XLU", "XLV", "XLY", "XLRE"],
    "anchors": ["NVDA", "TSLA", "AAPL", "AMD", "AMZN", "META",
                "MSFT", "GOOGL", "PLTR", "COIN", "SMCI"]
  }
}

GET /api/rate-status

Current backtest rate-limit counters for the caller's IP, without consuming a slot. Used by the homepage to render the "you have N left" banner on page load.

{
  "hourRemaining": 2,
  "dayRemaining":  5,
  "hourResetAt":   "2026-05-18T06:30:00.000Z",
  "dayResetAt":    "2026-05-19T05:30:00.000Z"
}

GET /api/chat-status

Current chat rate-limit counters and the ANTHROPIC_API_KEY liveness flag. If the key is unset, the chat panel still renders but every send returns 503.

{
  "hourRemaining": 30,
  "dayRemaining":  100,
  "hourResetAt":   "...",
  "dayResetAt":    "...",
  "available":     true
}

Write endpoints (rate-limited)

POST /api/backtest

Run one strategy on one symbol for a date range. Synchronous dispatcher: computes the content hash, short- circuits on cache hit, otherwise consumes one rate-limit slot and fires the background function.

Request

{
  "symbol":    "SPY",
  "startDate": "2024-01-01",
  "endDate":   "2025-12-31",
  "strategy":  {
    "kind": "sma_crossover",
    "fastWindow": 20,
    "slowWindow": 50
  }
}

Response (cache hit)

{
  "status": "ready",
  "hash":   "<sha256 hex>",
  "cached": true,
  "result": { /* see Result shape below */ }
}

Response (queued)

{
  "status": "queued",
  "hash":   "<sha256 hex>",
  "rateLimits": { /* updated info */ }
}

Response (429 rate-limited)

{
  "status":  "rate_limited",
  "reason":  "hour_exceeded" | "day_exceeded",
  "rateLimits": { /* counters + reset times */ }
}

POST /api/compare

Run all six strategies on one symbol over one date range. Consumes one rate-limit slot regardless of the internal backtest count. Cache key is the symbol + date range + strategy catalog hash (a different catalog version invalidates).

Request

{
  "symbol":    "XLE",
  "startDate": "2024-01-01",
  "endDate":   "2025-12-31"
}

Result shape (when ready)

{
  "symbol": "XLE",
  "startDate": "2024-01-01",
  "endDate": "2025-12-31",
  "results": [
    {
      "strategy": "buy_and_hold",
      "totalReturn": 0.234,
      "sharpe": 1.12,
      "maxDrawdown": 0.083,
      "hitRate": 0.534,
      "barCount": 504
    },
    /* ... five more strategies ... */
  ]
}

POST /api/scan

Run one strategy across all 23 universe symbols for one date range. Consumes one rate-limit slot. Internally fans out to one Supabase query with an IN clause covering all 23 symbols, then runs the WASM strategy 23 times against the partitioned bars.

Request

{
  "strategy":  { "kind": "momentum", "lookback": 60 },
  "startDate": "2024-01-01",
  "endDate":   "2025-12-31"
}

Result shape (when ready)

{
  "strategy": { "kind": "momentum", "lookback": 60 },
  "startDate": "2024-01-01",
  "endDate": "2025-12-31",
  "results": [
    {
      "symbol": "NVDA",
      "totalReturn": 0.847,
      "sharpe": 1.83,
      "maxDrawdown": 0.214,
      "hitRate": 0.521,
      "barCount": 504
    },
    /* ... 22 more symbols ... */
  ]
}

POST /api/chat

SelectBot. Server-Sent Events streaming response from claude-sonnet-4-6 via the Anthropic SDK. Cached system prompt covers Rust, this site, quant finance basics, and backtesting philosophy. Consumes one chat rate-limit slot per send.

Request

{
  "messages": [
    { "role": "user", "content": "What's Wilder's RSI recurrence?" }
  ]
}

Response

Content-Type text/event-stream. SSE events emit incremental delta messages matching the Anthropic streaming-event shape; clients should concatenate text_delta blocks to assemble the reply.

Response (503 if ANTHROPIC_API_KEY unset)

{
  "error": "chat_unavailable",
  "reason": "ANTHROPIC_API_KEY not set on the Netlify project"
}

Poll endpoint (no rate limit)

GET /api/result?hash=<hex>

Fetch a backtest, compare, or scan result by content hash. Returns 202 with { status: "pending" } while the background function is still running; returns 200 with the full result blob once it's complete.

Response (pending)

{ "status": "pending", "hash": "<hex>" }

Response (ready)

{
  "status": "ready",
  "hash":   "<hex>",
  "result": { /* the appropriate result shape from above */ }
}

Result shape (single backtest)

The /api/backtest result blob, also produced by the WASM core for use inside /api/compare and /api/scan aggregates.

{
  "symbol":      "SPY",
  "startDate":   "2024-01-01",
  "endDate":     "2025-12-31",
  "strategy":    { "kind": "sma_crossover", "fastWindow": 20, "slowWindow": 50 },
  "barCount":    504,
  "totalReturn": 0.187,
  "cagr":        0.093,
  "sharpe":      1.04,
  "maxDrawdown": 0.072,
  "hitRate":     0.542,
  "equity":      [ 1.0, 1.002, 1.001, /* ... */ ],
  "benchmark": {
    "label":       "buy and hold",
    "totalReturn": 0.211,
    "sharpe":      1.18,
    "equity":      [ 1.0, /* ... */ ]
  }
}

Strategy catalog

Every strategy's kind tag plus its parameter names. Defaults match the values returned by the Rust strategy_catalog WASM export.

buy_and_hold        (no params)
sma_crossover       fastWindow: u32, slowWindow: u32
momentum            lookback: u32
rsi_mean_reversion  period: u32, oversold: f64, overbought: f64
breakout            window: u32
bollinger_bands     period: u32, stdMultiplier: f64

Status codes

200 OK
Read endpoints, cached backtest result, finished result poll.
202 Accepted
Write endpoints when the background function was dispatched; poll /api/result.
400 Bad Request
Malformed JSON body or invalid params (e.g., unknown strategy kind, negative window).
429 Too Many Requests
Rate-limit exhausted. Response body carries reset timestamps; Retry-After header set in seconds.
500 Internal Server Error
Background function crashed before writing a result blob. The result blob will eventually carry an error field if the failure was downstream of the dispatcher.
503 Service Unavailable
/api/chat only, when ANTHROPIC_API_KEY is unset on the Netlify project.
Try it. curl https://selectsectors.com/api/health returns the live catalog. The version, deployed commit, and strategy list let you sanity-check that the deploy you're hitting is the deploy you expect.
